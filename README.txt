Порядок использования:
- запустить integral.py
- ввести выражение в формате языка Python (можно использовать функции из модуля math)
- ввести левую и правую границы для графика функции, частоту дискретизации
- получить график интеграла этой функции, выполненный с помощью библиотеки matplotlib
- при желании можно экспортировать график как изображение
